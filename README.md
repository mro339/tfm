## **Comandos básicos ejecución**
Una pequeña guía para ejecutar el modelo. Todo estos comandos se realizan en la terminal:

1. Para generar el script "docker-compose.yml"
```
pyhton docker-compose.yml
```
2. Para levantar y ejecutar contenedores (Clientes y servidor):
En la terminal:
```
docker compose up --build
```
3. Para bajar los contenedores
```
docker compose down
```
